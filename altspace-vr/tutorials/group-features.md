# <a name="using-the-groups-feature-to-create-private-events"></a>使用组功能创建私有事件

我们的组功能为主机提供了创建被批准的来宾列表的 VR 空间的能力。 主机可以手动添加要向其空间授予访问权限的任何 AltspaceVR 用户名/电子邮件。 下面介绍了如何使用此功能来托管事件、会议和会议：

> [!NOTE]
> 在访问 "组" 之前，你需要通过登录到 altvr.com，在网站上编辑用户配置文件并选择 "早期访问" 来选择使用早期访问功能。 或在 AltspaceVR 应用程序（"设置"、"常规"）中参与早期访问计划。

1. 在浏览器中，访问  [https://altvr.com](https://altvr.com)   并登录到你的帐户。 
2. 查找 **更多** 下拉菜单。
3. 选择 **组**

![突出显示更多下拉菜单的 AltspaceVR web 视图](images/group-features-img-01.png)

4. 在 "组" 页中，你可以看到你所属的组的列表或的管理员。 

![AltspaceVR web 视图组页](images/group-features-img-02.png)

5. 选择 " **创建** " 按钮以创建新组。
6. 填写字段以创建组。

![具有输入字段的新建组创建页](images/group-features-img-03.png)

> [!NOTE]
> 您可以将鼠标悬停在绿色的 "？" 获取有关包含它们的字段的其他信息。

7. 向所有与会者发送电子邮件，并让他们使用同一电子邮件地址创建 AltspaceVR 帐户。 他们可以在世界上或在我们的 [网站](https://account.altvr.com/users/sign_up)上创建 AltspaceVR 帐户。

## <a name="how-to-create-a-private-eventfrom-a-group"></a>如何从组创建私有事件 

1. 在 "成员" 或 "管理员" 页中，选择要访问的组。 
2. **作为管理员**，你可以修改你以前配置的所有选项以创建组。请访问组页面上的 "编辑" 按钮以访问它们。  
3. **为组添加其他管理员：**  编辑时，可以将其他用户添加为管理员来帮助管理组。  
    * 这必须是用户的 AltspaceVR 用户名，而不是其帐户电子邮件。  
4. 选择 " **创建事件**"。  

![已创建事件](images/group-features-img-04.png)

3. 在顶部查看 **我的事件**
4. 使用磁贴图像，按主视图中的事件创建者提供的 **名称** 查找事件。选择它以输入事件（如果它是活动的）。

## <a name="how-to-use-patreon-and-eventbrite-to-host-a-ticketed-vr-event"></a>如何使用 Patreon 和 Eventbrite 来托管 Ticketed VR 事件： 

可使用 Patreon、Eventbrite 在 Altspace 中创建 ticketed 事件。你需要创建用户的电子邮件地址列表，或从你的付款平台导出该信息。  

### <a name="creating-a-list"></a>创建列表

使用电子表格程序创建包含所有电子邮件地址的单个列。 完成此操作后，请执行以下操作：  

1. 创建或编辑组。  
2. 查找 " **成员电子邮件** " 字段。  
3. 找到 " **选择文件** " 选项以从 * .csv 文件导入电子邮件。
4. 应导入列表，每行包含一个电子邮件。

> [!NOTE]
> 如果选择，还可以将电子邮件复制并粘贴到字段，而不是从 * .csv 文件导入。

### <a name="importing-a-list"></a>导入列表

使用 Patreon 和 Eventbrite 这样的平台，你可以将 * .csv 文件 (以逗号分隔的值) ，其中包含专用的 "电子邮件" 列。您可以选择此列以供使用。完成此操作后，步骤几乎是相同的，它们是：  

1. 创建或编辑组。  
2. 查找 " **成员电子邮件** " 字段。  
3. 找到 " **选择文件** " 选项以从 * .csv 文件导入电子邮件。
4. 应导入列表，每行包含一个电子邮件。

> [!NOTE]
> 如果选择，还可以将电子邮件复制并粘贴到字段，而不是从 * .csv 文件导入。

## <a name="faq"></a>常见问题解答 

### <a name="im-having-trouble-filling-out-the-fields-in-groups-what-do-they-mean"></a>在组中填写字段时遇到问题。 它们是什么意思？

 ( * 可以将鼠标悬停在绿色的 "？"获取有关包含它们的字段的附加信息。 )  

* **名称**  –将对所有成员可见的组的名称。   
* **描述**  –用于说明组用途的字段。 对所有成员可见。
* **用户**  –此字段允许你列出组的组成员作为用户名或电子邮件地址，只要它是每行一个。这是一种定义组成员的方法。 **但是，对于使用外部支付平台（如 Patreon）的人员，请使用此页后面的 "电子邮件允许列表" 字段。**
    * 每行填写一次用户名或电子邮件。  
    * 若要在 AltspaceVR 中查找用户名，请让用户打开 "主菜单" 并转到 "Me" 部分。它是括在括号中的名称。  
    * 若要在网站上查找用户名，请让用户从 "> 配置文件" 选项中转到 "配置文件"。 可以在 "配置文件" 页的左栏中找到该文件。  
    * 此字段区分大小写，因此 "username" 和 "Username" 被视为不同。  
    * 此字段接受与有效帐户关联的用户名和电子邮件。请确保每行有一项。如果电子邮件或用户名是同一人，则不需要将其放在一起。  
    * 帐户必须处于良好地位。  
    * 可以通过浏览器的 "Ctrl + F" 的 "查找" 选项搜索列表。  
* **图像**  –用作组的促销横幅图像。它在组网页和 VR 中可见。  
    * 建议为 1920 x1080，以获得更好的图像保真度。  
    * 建议为 JPEG 或 PNG。  
    * 图像必须为16:9 纵横比。  
    * 我们将从上传中自动生成缩略图，并在 AltspaceVR 和网站的各种 UI 屏幕中使用它们。  
* **配置文件映像**  –用作组配置文件映像。 它在组网页和 VR 中可见。  
    * 建议512×512以获得更好的图像保真。  
    * 建议为 JPEG 或 PNG。  
    * 我们将从上传中自动生成缩略图，并在 AltspaceVR 和网站的各种 UI 屏幕中使用它们。  
* **选择环境**  –位于页面右侧，你将看到要使用的可用环境的各种缩略图。 此环境将充当你的组的主要 hangout，并将在 "名称" 字段中显示为你和你的组成员，并显示在 "Commons"> 字段中的 "> 世界-> <名称"。  
    * 如果你想要创建自己的环境，则必须通过 [网站](https://account.altvr.com/worlds/my)执行此操作，你将看到这些世界是可能的选项。 

> [!NOTE]
> 此功能以 Beta 版本提供。 我们鼓励组在 "环境选择" 对话框中利用预先存在的世界。

* "Commons" 一词将始终显示为任何组的 hangout 世界的后缀。  
* **成员电子邮件**  –此字段允许你按与 AltspaceVR 帐户关联的电子邮件列出用户。 **使用此列表将覆盖上述 "用户" 列表。**  这是管理组成员身份的建议方法，特别是使用外部支付平台的成员身份。  
    * 填写此字段时，请确保 AltspaceVR 帐户中列出了相关电子邮件，否则可能会失去对组的访问权限。  
    * 将地址粘贴到字段中时，请确保每行有一封电子邮件。  
    * 此列表接受没有有效 AltspaceVR 帐户的电子邮件。AltspaceVR 用户名经过验证，电子邮件地址不会。 但是，当该电子邮件用于创建 AltspaceVR 帐户时，与之关联的用户将获得对该组的访问权限。  
    * 可从 * .csv 文件导入列表。 选择 "选择文件" 选项，然后选择文件。
    * 我们建议使用成员电子邮件来管理从付款平台（如 Patreon 或 Eventbrite）创建的组，因为它是一种更好的方式来协调您的成员。请注意，组管理员需要通知其用户/成员，他们必须使用他们在支付平台上使用的相同电子邮件地址来创建 AltspaceVR 帐户。  
    * 有关如何获取用户列表的说明，请参阅将用户导入组部分。   
* **启用自动导入**  –自动导入是在电子邮件允许列表中运行的重复过程。 它协调电子邮件现在是否具有与之关联的有效 AltspaceVR 帐户并授予访问权限。   
    * 如果使用 csv 文件上传，建议启用此方法。
* **YouTube 视频 ID**  –允许你将 YouTube 视频嵌入到网站上的组登录页。请注意，这仅适用于组网站，且不会出现在 AltspaceVR 中。
    * 必须使用视频 id，而不是完整地址。 视频 id 是 "=" 后出现的一组字符， (等号) 登录 YouTube。  
* **Twitter 句柄**  -将最新的推文从 Twitter 帐户嵌入到网站上的组登录页。请注意，这仅适用于组网站，且不会出现在 AltspaceVR 中。
* **创建组**  –位于页面底部的按钮。 完成上述字段后，选择此按钮以创建组。

### <a name="i-want-to-use-eventbrite-and-patreon-but-i-dont-see-where-to-get-the-email-addresses-for-my-groups"></a>我要使用 Eventbrite 和 Patreon，但不会看到在何处获取我的组的电子邮件地址

**Patreon ** 

[Patreon](https://www.patreon.com/)  是成员身份平台，creatives 可在其中为其内容创建订阅计划。尽管 Patreon 可以通过多种方式来导出 pledge 数据，但我们将重点介绍 CSV 方向。 您可以在 [此处](https://www.patreon.com/portal/how-to/export-pledge-data)找到此信息的帮助页，也   可以按照以下说明进行操作：  

1. 在 Patreon 站点左侧的创建者侧栏中查找 "顾客" 字段。  
2. 展开它以查找 "关系管理器" 并选择它。  
3. 页面顶部是一个名为 "CSV" 的 "下载" 按钮。 选择该文件夹。  
4. 它会弹出一个 windows 对话框字段以保存该文件。保存它。  
5. 按照 "AltspaceVR 组" 页中的导入说明使用此文件。  

**Eventbrite ** 

[Eventbrite](https://www.eventbrite.com)  是一个票证网站，允许用户管理和创建事件的票证。与 Patreon 一样，可以通过多种方式生成与会者电子邮件列表。再次强调 CSV 方向。 您可以在 [此处](https://www.eventbrite.com/support/articles/en_US/How_To/how-to-edit-and-export-an-attendee-summary-report?lg=en_US)找到此信息的帮助页，   也可以按照下面的说明进行操作以获取此数据的一种方法。  

1. 在配置文件菜单中，找到 "管理事件" 菜单。 
2. 选择要在其中获取与会者电子邮件列表的事件。 
3. 在事件的 "仪表板" 中，找到名为 "通用报表" 的类别。 
4. 还可以在 "分析-> 事件报告" 下的左侧列菜单中找到 "常见报告"   
5. 选择 "与会者摘要"。 
6. 查找 "报表结果" 部分。 
7. 选择 "CSV" 按钮以下载该文件。 

> [!IMPORTANT]
> 这是一项 beta 版功能。 AltspaceVR 是免费体验，不负责可能会阻止与会者访问 ticketed 事件的任何技术问题。
