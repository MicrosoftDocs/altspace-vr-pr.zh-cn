# <a name="using-the-groups-feature-to-create-private-events"></a>使用组功能创建专用事件

我们的组功能使主机能够创建向已批准的来宾列表开放的 VR 空间。 主机可以手动添加任何要授予其空间访问权限的 AltspaceVR 用户名/电子邮件。 下面将说明如何使用此功能来托管活动、会议与会议：

> [!NOTE]
> 在可以访问"组"之前，需要通过登录到 altvr.com、编辑网站的用户配置文件并选择"提前访问"来选择加入早期访问功能。 或者，在 AltspaceVR 应用中，设置"常规"和"参与早期访问计划"。

1. 在浏览器中，转到  [https://altvr.com](https://altvr.com)   并登录到帐户。 
2. 找到 **"更多** "下拉菜单。
3. 选择 **组**

![AltspaceVR Web 视图，突出显示了更多下拉菜单](images/group-features-img-01.png)

4. 进入"组"页后，可以看到你属于或其管理员的组列表。 

![AltspaceVR Web 视图组页](images/group-features-img-02.png)

5. 在"创建 **"** 按钮上选择以创建新组。
6. 填写字段以创建组。

![包含输入字段的新组创建页](images/group-features-img-03.png)

> [!NOTE]
> 可以将鼠标悬停在绿色的"？" ，获取有关具有这些字段的字段的其他信息。

7. 向所有与会者发送电子邮件，让他们使用相同的电子邮件地址创建 AltspaceVR 帐户。 他们可以创建世界上的 AltspaceVR 帐户，也可以在我们的网站上 [创建](https://account.altvr.com/users/sign_up)。

## <a name="how-to-create-a-private-eventfrom-a-group"></a>如何从组创建专用事件 

1. 进入"成员"或"管理员"页后，选择要访问的组。 
2. **作为管理员**，你可以修改之前配置的所有选项以创建组。转到组页面上的"编辑"按钮以访问它们。  
3. **为组添加其他管理员：**  编辑时，可以将其他用户添加为管理员，以帮助管理组。  
    * 这必须是用户的 AltspaceVR 用户名，而不是其帐户电子邮件。  
4. 选择“创建事件”。  

![已创建事件](images/group-features-img-04.png)

## <a name="how-to-use-patreon-and-eventbrite-to-host-a-ticketed-vr-event"></a>如何使用 Patreon 和 Eventbrite 托管票证 VR 事件： 

可以使用 Patreon、Eventbrite 在 Altspace 中创建票证事件。需要创建用户的电子邮件地址列表，或者从支付平台导出该信息。  

### <a name="creating-a-list"></a>创建列表

使用电子表格程序创建包含所有电子邮件地址的单个列。 完成此操作后，执行以下操作：  

1. 创建或编辑组。  
2. 查找" **成员电子邮件"** 字段。  
3. 找到" **选择** 文件"选项，从 *.csv文件导入电子邮件。
4. 应每行导入一封电子邮件的列表。

> [!NOTE]
> 如果选择，还可以将电子邮件复制并粘贴到 字段中，而不是从 *.csv 文件导入。

### <a name="importing-a-list"></a>导入列表

Patreon 和 Eventbrite 等平台允许导出 *.csv 文件 (逗号分隔值) 具有专用的"电子邮件"列。可以选择此列供使用。提供此功能后，步骤主要相同，它们是：  

1. 创建或编辑组。  
2. 查找" **成员电子邮件"** 字段。  
3. 找到" **选择** 文件"选项，从 *.csv文件导入电子邮件。
4. 应每行导入一封电子邮件的列表。

> [!NOTE]
> 如果选择，还可以将电子邮件复制并粘贴到 字段中，而不是从 *.csv 文件导入。

### <a name="im-having-trouble-filling-out-the-fields-in-groups-what-do-they-mean"></a>在组中填写字段时遇到问题。 它们是什么意思？

 (*可以将鼠标悬停在绿色的"？"若要获取有关具有这些字段的字段的其他信息，)  

* **名称**  – 对所有成员可见的组的名称。   
* **说明**  - 用于描述组用途的字段。 对所有成员可见。
* **用户**  - 此字段允许你将组的成员作为用户名或电子邮件地址列出，只要每行一个。这是定义其成员的一种方法。 **但是，对于使用外部支付平台（如 Patreon）** 的人，请使用本页下部所述的"电子邮件允许列表"字段。
    * 每行填写用户名或电子邮件。  
    * 若要在 AltspaceVR 中查找用户名，请让用户打开"主菜单"并转到"Me"部分。它是括号中的名称。  
    * 若要在网站上查找用户名，请让用户从"更多 ->配置文件"选项转到"配置文件"。 可以在"配置文件"页的左侧列中找到它。  
    * 字段区分大小写，因此"username"和"Username"被视为不同。  
    * 此字段接受与有效帐户关联的用户名和电子邮件。确保每行有一项。如果电子邮件或用户名是同一个人，则无需同时输入，这两者均能正常工作。  
    * 帐户必须良好。  
    * 可以使用浏览器"Ctrl + F"的"查找"选项搜索列表  
* **图像**  – 用作组的促销横幅图像。它在组网页和 VR 中可见。  
    * 建议使用 1920 x1080 提高图像保真度。  
    * 首选 JPEG 或 PNG。  
    * 图像必须是 16：9 纵横比。  
    * 我们自动生成上传的缩略图，并将其用于 AltspaceVR 和网站的各种 UI 屏幕。

    * 建议使用 512×512 提高图像保真度。  
    * 首选 JPEG 或 PNG。  
    * 我们自动生成上传的缩略图，并将其用于 AltspaceVR 和网站的各种 UI 屏幕。  
* **选择环境**  – 位于页面右侧，你将看到可用的环境的各种缩略图。 此环境将充当组的主要注销"世界"，并且会显示在"名称"字段"> Commons"中的"Worlds -> My Worlds -> <Name"中。  
    * 如果要创建自己的世界，则必须通过网站 进行创建，你将看到这些[](https://account.altvr.com/worlds/my)世界作为可能的选项。

> 此功能以 Beta 版本提供。 我们鼓励组在 "环境选择" 对话框中利用预先存在的世界。

* "Commons" 一词将始终显示为任何组的 hangout 世界的后缀。  
* **成员电子邮件**  –此字段允许你按与 AltspaceVR 帐户关联的电子邮件列出用户。 **使用此列表将覆盖上述 "用户" 列表。**  这是管理组成员身份的建议方法，特别是使用外部支付平台的成员身份。  
    * 填写此字段时，请确保 AltspaceVR 帐户中列出了相关电子邮件，否则可能会失去对组的访问权限。  
    * 将地址粘贴到字段中时，请确保每行有一封电子邮件。  
    * 此列表接受没有有效 AltspaceVR 帐户的电子邮件。AltspaceVR 用户名经过验证，电子邮件地址不会。 但是，当该电子邮件用于创建 AltspaceVR 帐户时，与之关联的用户将获得对该组的访问权限。  
    * 可以从 * .csv 文件导入列表。 选择 "选择文件" 选项，然后选择文件。
    * 我们建议使用成员电子邮件来管理从付款平台（如 Patreon 或 Eventbrite）创建的组，因为它是一种更好的方式来协调您的成员。请注意，组管理员需要通知其用户/成员，他们必须使用他们在支付平台上使用的相同电子邮件地址来创建 AltspaceVR 帐户。  
    * 有关如何获取用户列表的说明，请参阅将用户导入组部分。   
* **启用自动导入**  –自动导入是在电子邮件允许列表中运行的重复过程。 它协调电子邮件现在是否具有与之关联的有效 AltspaceVR 帐户并授予访问权限。   
    * 如果使用 csv 文件上传，建议启用此方法。

    * 必须使用视频 id，而不是完整地址。 视频 id 是 "=" 后出现的一组字符， (等号) 登录 YouTube。  
* **Twitter 句柄**  -将最新的推文从 Twitter 帐户嵌入到网站上的组登录页。请注意，这仅适用于组网站，且不会出现在 AltspaceVR 中。
* **创建组**  –位于页面底部的按钮。 完成上述字段后，选择此按钮以创建组。

### <a name="i-want-to-use-eventbrite-and-patreon-but-i-dont-see-where-to-get-the-email-addresses-for-my-groups"></a>我要使用 Eventbrite 和 Patreon，但不会看到在何处获取我的组的电子邮件地址

**Patreon ** 

[Patreon](https://www.patreon.com/)  是成员身份平台，creatives 可在其中为其内容创建订阅计划。尽管 Patreon 可以通过多种方式来导出 pledge 数据，但我们将重点介绍 CSV 方向。 您可以在 [此处](https://www.patreon.com/portal/how-to/export-pledge-data)找到此信息的帮助页，也   可以按照以下说明进行操作：  

1. 在 Patreon 站点左侧的创建者侧栏中查找 "顾客" 字段。  
2. 展开它以查找 "关系管理器" 并选择它。  
3. 页面顶部是一个名为 "CSV" 的 "下载" 按钮。 选择该文件夹。  
4. 它会弹出一个 windows 对话框字段以保存该文件。保存它。  
5. 按照 "AltspaceVR 组" 页中的导入说明使用此文件。  

**Eventbrite ** 

[Eventbrite](https://www.eventbrite.com)  是一个票证网站，允许用户管理和创建事件的票证。与 Patreon 一样，可以通过多种方式生成与会者电子邮件列表。再次强调 CSV 方向。 您可以在 [此处](https://www.eventbrite.com/support/articles/en_US/How_To/how-to-edit-and-export-an-attendee-summary-report?lg=en_US)找到此信息的帮助页，   也可以按照下面的说明进行操作以获取此数据的一种方法。  

1. 在配置文件菜单中，找到 "管理事件" 菜单。 
2. 选择要在其中获取与会者电子邮件列表的事件。 
3. 在事件的 "仪表板" 中，找到名为 "通用报表" 的类别。 
4. 还可以在 "分析-> 事件报告" 下的左侧列菜单中找到 "常见报告"   
5. 选择 "与会者摘要"。 
6. 查找 "报表结果" 部分。 
7. 选择 "CSV" 按钮以下载该文件。 

> [!IMPORTANT]
> 这是一项 beta 版功能。 AltspaceVR 是免费体验，不负责可能会阻止与会者访问 ticketed 事件的任何技术问题。
