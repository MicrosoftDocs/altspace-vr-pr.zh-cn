# <a name="using-the-groups-feature-to-create-private-events"></a>使用组功能创建专用事件

我们的组功能使主机能够创建向已批准的来宾列表开放的 VR 空间。 主机可以手动添加任何要授予其空间访问权限的 AltspaceVR 用户名/电子邮件。 下面将说明如何使用此功能来托管活动、会议与会议：

> [!NOTE]
> 在可以访问"组"之前，需要通过登录到 altvr.com、编辑网站的用户配置文件并选择"提前访问"来选择加入早期访问功能。 或者，在 AltspaceVR 应用中，设置"常规"和"参与早期访问计划"。

1. 在浏览器中，转到  [https://altvr.com](https://altvr.com)   并登录到帐户。 
2. 找到 **"更多** "下拉菜单。
3. 选择 **组**

![AltspaceVR Web 视图，突出显示了更多下拉菜单](images/group-features-img-01.png)

4. 进入"组"页后，可以看到你属于或其管理员的组列表。 

![AltspaceVR Web 视图组页](images/group-features-img-02.png)

5. 在"创建 **"** 按钮上选择以创建新组。
6. 填写字段以创建组。

![包含输入字段的新组创建页](images/group-features-img-03.png)

> [!NOTE]
> 可以将鼠标悬停在绿色的"？" ，获取有关具有这些字段的字段的其他信息。

7. 向所有与会者发送电子邮件，让他们使用相同的电子邮件地址创建 AltspaceVR 帐户。 他们可以创建世界上的 AltspaceVR 帐户，也可以在我们的网站上 [创建](https://account.altvr.com/users/sign_up)。

## <a name="how-to-create-a-private-eventfrom-a-group"></a>如何从组创建专用事件 

1. 进入"成员"或"管理员"页后，选择要访问的组。 
2. **作为管理员**，你可以修改之前配置的所有选项以创建组。转到组页面上的"编辑"按钮以访问它们。  
3. **为组添加其他管理员：**  编辑时，可以将其他用户添加为管理员，以帮助管理组。  
    * 这必须是用户的 AltspaceVR 用户名，而不是其帐户电子邮件。  
4. 选择“创建事件”。  

![已创建事件](images/group-features-img-04.png)

## <a name="how-to-attend-a-group-event-as-an-invitee"></a>如何作为受邀者参加组活动

![主 AltspaceVR 应用菜单](images/group-features-img-05.png)

3. 在顶部，查找" **我的事件"**
4. 使用磁贴图像按 **事件** 创建者在主视图中提供的名称查找事件。选择它以输入活动（如果活动）。

![AltspaceVR 应用，突出显示了"我的事件"面板](images/group-features-img-06.png)

## <a name="how-to-use-patreon-and-eventbrite-to-host-a-ticketed-vr-event"></a>如何使用 Patreon 和 Eventbrite 托管票证 VR 事件： 

可以使用 Patreon、Eventbrite 在 Altspace 中创建票证事件。需要创建用户的电子邮件地址列表，或者从支付平台导出该信息。  

### <a name="creating-a-list"></a>创建列表

使用电子表格程序创建包含所有电子邮件地址的单个列。 完成此操作后，执行以下操作：  

1. 创建或编辑组。  
2. 查找" **成员电子邮件"** 字段。  
3. 找到" **选择** 文件"选项，从 *.csv文件导入电子邮件。
4. 应每行导入一封电子邮件的列表。

> [!NOTE]
> 如果选择，还可以将电子邮件复制并粘贴到 字段中，而不是从 *.csv文件。

### <a name="importing-a-list"></a>导入列表

Patreon 和 Eventbrite 等平台允许导出 *.csv 文件 (逗号分隔值) 具有专用"电子邮件"列。可以选择此列供使用。提供此功能后，步骤主要相同，它们是：  

1. 创建或编辑组。  
2. 查找" **成员电子邮件"** 字段。  
3. 找到" **选择** 文件"选项，从 *.csv文件导入电子邮件。
4. 应每行导入一封电子邮件的列表。

> [!NOTE]
> 如果选择，还可以将电子邮件复制并粘贴到 字段中，而不是从 *.csv文件。

## <a name="faq"></a>常见问题解答 

### <a name="im-having-trouble-filling-out-the-fields-in-groups-what-do-they-mean"></a>在组中填写字段时遇到问题。 它们是什么意思？

 (*可以将鼠标悬停在绿色的"？"若要获取有关具有这些字段的字段的其他信息，)  

* **名称**  – 对所有成员可见的组的名称。   
* **说明**  - 用于描述组用途的字段。 对所有成员可见。
* **用户**  - 此字段允许你将组的成员作为用户名或电子邮件地址列出，只要每行一个。这是定义其成员的一种方法。 **但是，对于使用外部支付平台（如 Patreon）** 的人，请使用本页下部所述的"电子邮件允许列表"字段。
    * 每行填写用户名或电子邮件。  
    * 若要在 AltspaceVR 中查找用户名，请让用户打开"主菜单"并转到"Me"部分。它是括号中的名称。  
    * 若要在网站上查找用户名，请让用户从"更多 ->配置文件"选项转到"配置文件"。 可以在"配置文件"页的左侧列中找到它。  
    * 字段区分大小写，因此"username"和"Username"被视为不同。  
    * 此字段接受与有效帐户关联的用户名和电子邮件。确保每行有一项。如果电子邮件或用户名是同一个人，则无需同时输入，这两者均能正常工作。  
    * 帐户必须良好。  
    * 可以使用浏览器"Ctrl + F"的"查找"选项搜索列表  
* **图像**  – 用作组的促销横幅图像。它在组网页和 VR 中可见。  
    * 建议使用 1920 x1080 提高图像保真度。  
    * 首选 JPEG 或 PNG。  
    * 图像必须是 16：9 纵横比。  
    * 我们自动生成上传的缩略图，并将其用于 AltspaceVR 和网站的各种 UI 屏幕。  
* **配置文件图像**  – 用作组配置文件映像。 它在组网页和 VR 中可见。  
    * 建议使用 512×512 提高图像保真度。  
    * 首选 JPEG 或 PNG。  
    * 我们自动生成上传的缩略图，并将其用于 AltspaceVR 和网站的各种 UI 屏幕。  
* **选择环境**  – 位于页面右侧，你将看到可用的环境的各种缩略图。 此环境将充当组的主要环比世界，并且会显示在"名称"字段中的"Worlds -> My Worlds -> <Name"中> Commons"中。  
    * 如果要创建自己的世界，则必须通过网站 进行创建，你将看到这些[](https://account.altvr.com/worlds/my)世界作为可能的选项。 

> [!NOTE]
> 此功能以 Beta 版本提供。 我们鼓励组在"环境选择"对话框中利用预先不存在的"世界"。

* "Commons"一词始终作为后缀出现在任何组的环比世界。  
* **成员电子邮件**  - 此字段允许通过与 AltspaceVR 帐户关联的电子邮件列出用户。 **使用此列表将替代上述"用户"列表。**  这是管理组成员身份（尤其是使用外部支付平台的组）的建议方法。  
    * 填写此字段时，请确保 AltspaceVR 帐户关联的电子邮件已列出，否则可能会失去对组的访问权限。  
    * 将地址粘贴到其字段中时，请确保每行有一封电子邮件。  
    * 此列表接受没有有效 AltspaceVR 帐户的电子邮件。AltspaceVR 用户名已验证，电子邮件地址不会验证。 但是，当该电子邮件用于创建 AltspaceVR 帐户时，与其关联的用户将获得对组的访问权限。  
    * 可以从 *.csv 文件导入列表。 选择"选择文件"选项，然后选择文件。
    * 建议使用"成员电子邮件"作为管理从 Patreon 或 Eventbrite 等支付平台创建的组的方法，因为这是一种更明确的协调成员的方式。请注意，组管理员需要通知其用户/成员，他们必须使用在支付平台上使用的同一确切电子邮件地址来创建 AltspaceVR 帐户。  
    * 请参阅将用户导入组部分，查看有关如何获取用户列表的说明。   
* **启用自动导入**  – 自动导入是在电子邮件"允许列表"中运行的重复过程。 它协调电子邮件现在是否具有与其关联的有效 AltspaceVR 帐户，并授予访问权限。   
    * 如果使用 csv 文件上传，建议启用它。
* **YouTube 视频 ID**  – 允许将 YouTube 视频嵌入网站的组登陆页面。请注意，这仅适用于组网站，不会显示在 AltspaceVR 中。
    * 必须使用视频 ID，而不是完整地址。 视频 ID 是在登录 YouTube 的"=" (后) 的字符集。  
* **Twitter 句柄**  - 将 Twitter 帐户中的最新推文嵌入到网站的组登陆页上。请注意，这仅适用于组网站，不会显示在 AltspaceVR 中。
* **创建组**  – 页面底部的按钮。 完成上述字段后，选择此按钮以创建组。

### <a name="i-want-to-use-eventbrite-and-patreon-but-i-dont-see-where-to-get-the-email-addresses-for-my-groups"></a>我想要使用 Eventbrite 和 Patreon，但看不到在何处获取组的电子邮件地址

**Patreon ** 

[Patreon](https://www.patreon.com/)  是一个成员资格平台，创意者可以在其中为其内容创建订阅计划。虽然 Patreon 有多种导出数据的方法，但我们将重点介绍 CSV 方向。 可在此处找到此信息的帮助 [页](https://www.patreon.com/portal/how-to/export-pledge-data)   ，也可以按照以下说明操作：  

1. 在 Patreon 站点左侧的创建者侧栏中找到"用户"字段。  
2. 展开它以找到"关系管理器"并选择它。  
3. 页面顶部是名为 CSV 的下载按钮。 选择该文件夹。  
4. 它将弹出一个窗口对话框字段以保存文件。保存它。  
5. 按照 AltspaceVR 组页中的导入说明使用此文件。  

**Eventbrite ** 

[Eventbrite](https://www.eventbrite.com)  是一个票证网站，允许用户管理和创建活动的票证。与 Patreon 一样，通过多种方式生成参与者电子邮件列表。我们将再次重点介绍 CSV 方向。 可在此处找到此信息的帮助 [页](https://www.eventbrite.com/support/articles/en_US/How_To/how-to-edit-and-export-an-attendee-summary-report?lg=en_US)，也可以按照以下说明获取提取   此数据的一种方法。  

1. 在"配置文件"菜单中，找到"管理事件"菜单。 
2. 选择要获取参与者电子邮件列表的事件。 
3. 在事件的仪表板中，找到名为"常见报表"的类别。 
4. 还可以在左侧列菜单中的"分析 ->事件报告"下找到"常见报告"   
5. 选择"参与者摘要"。 
6. 找到"报告结果"部分。 
7. 选择"CSV"按钮以下载文件。 

> [!IMPORTANT]
> 这是 beta 版本功能。 AltspaceVR 是一种免费体验，不负责体验中可能出现的任何技术问题，这些问题可能会阻止与会者访问票证事件。
