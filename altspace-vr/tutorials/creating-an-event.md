# <a name="creating-an-event"></a>创建事件

这是在 AltspaceVR 中创建事件的分步指南。 强烈建议你在 AltspaceVR 中出席几个事件，以了解其工作原理。 检查 [AltspaceVR 事件日历](https://account.altvr.com/events) 以获取所有事件的列表。

本文内容：

* AltspaceVR 事件术语
* 创建事件和事件空间或世界
* 使用用于幻灯片演示的多媒体控制台
* 用图像标记事件
* 使用全球编辑器/工具包添加3D 对象
* 如何记录或实时流我的事件

## <a name="altspacevr-event-terminology"></a>AltspaceVR 事件术语

以下是需要熟悉的术语来创建事件和事件空间：

</br>

| 术语 | 定义 |
|---|---|
| 世界建筑 | AltspaceVR 提供了创建和自定义虚拟世界的能力。 AltspaceVR 托管支持文档、Discord 通道和全球事件，可帮助你 [了解有关如何获取帮助和开始生成的详细信息](../world-building/world-building-faq.md)。 |
| World | 世界是 AltspaceVR 中的虚拟空间。 它可能是办公室或大量的山地范围。 它是高度可自定义的环境。 它位于一个范围内，在该范围内可能有多个领域。 如果希望有多个事件空间用于特殊事件、培训中心和不同的会议空间来情趣，请将这两个领域添加到同一个 Universe，使其成为一个组。 |
| Universe | AltspaceVR world 构建 vernacular 中的宇宙代表了世界的分类。 每个 universe 可以拥有多个领域。 世界继承了宇宙的设置，使用户可以轻松地将人员添加到多个领域的允许列表和其他控制功能。 有关详细信息，请参阅 [管理您的世界](../world-building/managing-worlds.md) 。 |
| 模板 | 模板 (或空间模板) 是可以使用的预建世界或环境，而不是使用世界建筑功能创建一个。 AltspaceVR 为不同的体验和事件提供各种模板。 |
| 事件空间 | 事件空间是 AltspaceVR 中世界的同义词。 通常，它是指用来托管事件的世界。 |
| 网站 | 对 "网站" 的引用是 [AltspaceVR 的网站](https://altvr.com/)。 通常通过计算机或平板电脑上的 [事件网页](https://account.altvr.com/events/my) 创建和编辑事件，而不是通过 VR 设备来创建和编辑事件。 还需要访问网站以进行广泛的 [构建](../world-building/managing-worlds.md) 。 |
| 上下文角色 | [上下文角色](../getting-started/roles.md) 由事件创建者或世界生成器分配。 这些角色为用户提供了世界级或事件空间额外的特性和功能。 目前，这包括主机、表示器、版主、试点 (航班) 、Terraformer (world 构建) 和扩音器。 可以单独或全局分配这些角色，使每个人都可以在事件空间或世界中拥有相同的角色。 |
| 用户界面 (UI) /Menu | 当你位于世界各地的 AltspaceVR 中时，屏幕左侧和右侧会出现菜单。 带有 AltspaceVR 徽标的圆圈或主菜单将打开 (UI) 或菜单上的主用户界面，以访问不同的屏幕来浏览 AltspaceVR 和自定义你的体验。 可选的 UI 项显示在屏幕的正确大小，通常包括世界编辑器和主机工具。 通过使用光标单击它们来打开和交互。 |
| SDK/MRE | 这些是与软件开发工具包和 [混合现实扩展](../world-building/using-mixed-reality-extensions.md) 相关联的世界建筑术语，用于向世界建筑体验添加特性和功能。 这通常适用于更高级的用户。 |

## <a name="understanding-events"></a>了解事件

使用 AltspaceVR 可以轻松地创建一个世界，你可以使用预置的模板空间根据特定需求使用或自定义，或者从头开始创建一个世界。 本文将介绍如何使用预置的模板来创建事件。 以下各节介绍 [了如何使用图像对事件进行标记](#branding-your-event-with-images) ，以及如何 [使用世界编辑器和工具包添加3d 对象](#adding-3d-objects-using-world-editor-and-kits) 。 若要深入了解如何从头开始构建自定义世界，请在 AltspaceVR 中参加聚会的建设教程，并 [查看全球构建支持文档](../world-building/world-editor-getting-started.md)。

AltspaceVR 提供了两种方法来为事件创建空间。

* **一次性使用：** 创建一个事件并选择一个模板 world。
* **重复使用：** 创建一个世界空间，并将其导入该事件。

对于这两种情况，创建事件的过程是相同的，但有一个例外： [创建一个世界，然后将其导入](../world-building/managing-worlds.md) 为重复使用的事件空间。 还需要了解：

</br>

| 术语 | 定义 |
|---|---|
| 世界复制 | 为重复事件使用世界空间允许持续自定义。 签名、图像、生成点和其他自定义项将从事件保留到事件，而不是每次都自定义事件空间。 |
| 自定义事件世界 | 创建事件时，会将模板或导入世界复制并锁定到该事件中。 您可以对事件世界进行更改，而不会影响原始世界以及其他方法。 请考虑添加节日装饰、图片或其他装饰一次性使用世界编辑器元素，使事件空间更愉快，并适合事件。 |

除非更新了事件空间，否则对原始环境所做的修改不会处于事件世界中：
1. 使用事件网页上的 " **重新导入" 世界** 按钮。
2. 等待完成同步所需的2-3 分钟。 
3. 如果事件空间尚未更改，请参阅 " **设置" > 适中 > 重置空间** 以重置事件空间。 你需要将空间设置为世界生成器！

如果遇到技术问题，如未正确加载元素，请访问 AltspaceVR 菜单，并选择 " **设置" > 适中 > Reset Space** 重置事件空间并查看新的更改。 2D 中的 Windows PC 用户可以使用键盘快捷方式：在 AltspaceVR 中按 **CTRL + ALT + R** 可快速重置空间。

## <a name="creating-your-event-and-event-space-or-world"></a>创建事件和事件空间或世界

下面是有关为一次性事件或重复事件创建事件的分步说明。 选择模板或导入事件的世界。 

> [!NOTE]
> AltspaceVR 事件网页通过使用绿色问号按钮，提供窗体的每个方面的说明。 将光标滚动到它们上以获得特定说明。

在 AltspaceVR 网站上的 " [事件" > 我的事件](https://account.altvr.com/events/my) "网页上，选择" **计划事件** "或直接转到" AltspaceVR "网页中的" [创建事件](https://account.altvr.com/events/new)"。

1. **事件标题：** 键入事件的名称。 使其在 AltspaceVR 的网站和全球界面的各种日历视图中保持特定和简明。 尝试将标题保留为23个字符（包括单词之间的空格）。
2. **说明：** 键入事件的描述。
    * 说明中必须至少有10个字符，否则不会创建事件。
    * 在段落之间增加空白间距。
    * 若要添加指向事件 Facebook、Discord、网站或其他资源的链接，请使用以下格式 (此 Markdown 仅适用于网站上的事件升级页面，这在 AltspaceVR 菜单中无法正确呈现) ： `[Event Name](http://example.com/)`

3. **开始日期/结束日期：** 设置开始时间并确保结束时间晚到开始时间。
4. **类别：** 选择最能描述你的事件类型的类别。 
5. 将事件设置为 " **专用** " 或 " **公共**"。
    * 可见性：公共事件显示在 AltspaceVR 网站事件日历的 " [所有" 选项卡](https://account.altvr.com/events/all) 上，并公开到 "公共"。
    * 私有事件不显示在 AltspaceVR 事件日历上，并且需要事件 URL 来输入事件空间。
    * 如果希望 "添加为主事件"，则必须将事件设置为 "公共"。
6. **选择模板：** 网页右侧显示了 AltspaceVR 中可用模板的缩略图图像列表。 游戏机房、办公室、会议室、演示空白和有趣的会议空间。 选择一个看起来很有趣的。 如果你不喜欢，可以继续，只需使用新的模板创建一个新的事件。 如果要创建自己的自定义事件世界，请选择 " **冷** "，作为默认模板，然后按照以下说明进行操作，以导入您的世界。
7. 选择 **高级选项**

### <a name="advanced-options"></a>高级选项

1. **升级：** 每个事件都需要两个署名图像 (这些图像显示在 AltspaceVR 网站上，而不会出现在 AltspaceVR) 内的事件中：
    * **磁贴：** 磁贴图像需要为1920x1080 像素。 此图像将缩放到各种缩略图大小，并与其他事件一起显示在 AltspaceVR 事件日历中。 请确保图像清晰，并且没有太多文本。 不要使用您不拥有的版权所有的图像/徽标。
    * **横幅：** 需要1920x576 标题图像。 这将根据需要自动调整大小，并在事件信息或网页上可见。 半透明覆盖部分包含图像的较低25%。 避免在该区域中放置文本。
    * 另一 (更快) 选项是将 **磁贴图像复制到背景横幅图像**，这也会将磁贴图像用作标题图像。 请尝试一下： ) 。 如果看起来不好，可以创建新的横幅图像。
2. **在 VR 中：** 本部分包含的功能适用于在 AltspaceVR 应用内发生事件时的虚拟体验：
    * **默认上下文角色：** 绿色问号列出了 *事件中所有受众成员* 可用的特定角色。 最常见的是 *megaphone_only*。 添加此项可将 "主机工具" 按钮下的 "放大 My Voice" 选项添加到所有受众成员，因此，如果这是一个较大的空间，则可能会听到事件。 如果事件需要航班，请添加 *试点*。 若要添加两者，格式为： *megaphone_only，试点*。 每个用户需要通过转到 "设置"/"输入/动态" 在 AltspaceVR 应用程序中启用动态模式。
    * **说明：** 这些 [说明](../world-building/adding-welcome-messages.md) 适用于在空间中到达时生成欢迎图像的文本。 用户必须选择 "确定" 才能将其从视图中删除。 这通常用于提供访问群体说明，如 "请在受邀欢迎之前保持静音" 或 "欢迎使用我们将涵盖 ABC 的 XYZ 事件"。 这不是必需的，因此请谨慎使用。 还可以使用 Markdown 添加颜色和字体大小，更多详细信息： [http://digitalnativestudios.com/textmeshpro/docs/rich-text](http://digitalnativestudios.com/textmeshpro/docs/rich-text)
3. **高级：** 下面是有关事件的高级功能的快速说明 (在创建事件并编辑事件) 之前，某些功能将不会显示：
    * **管理员：** 管理员是你信任的 AltspaceVR 用户，可帮助你管理你的事件。 它们可能是 cohosts 或备份主机。 通过将其用户名添加到 Admins 列表，他们可以：
        * 更改事件标题、说明和其他功能。
        * 为版主、演示者和其他角色添加和删除上下文角色。
        * 删除事件。
    * **组：** 使用下拉菜单从专用 [组](group-features.md) 中进行选择。 仅当你的帐户已创建/已添加到组时才会显示。
    * **描述：** 这一简洁的句子将显示在事件网页上标题下面。
    * **从世界导入：** 若要使用已创建的世界事件空间重复使用，这是窗体中用于将该世界的布局和自定义导入到事件中的部分。 请注意以下事项：

    **导入您自己的环境：** 为了导入世界，你创建并拥有：
    * 选择下拉箭头以查看您创建的世界的列表。
    * 选择 "世界"。
    * 完成事件窗体的其余部分后，请在访问 AltspaceVR 中的事件空间前等待至少2分钟，以确保更新数据库信息并生成世界。
    
    **使用其他人：**
    * 与世界所有者联系，请求他们在个人世界上使用 " **好友共享** " 功能与你共享。

3. **YouTube 视频 ID：** 在事件的网页上可见，这会将 YouTube 事件尾端或视频添加到事件网页，而不是在世界中。 只需要 URL 中的视频 ID 部分： dQw4w9WgXcQ
4. **Twitter 句柄：** 这会将 Twitter 流添加到事件的网页中。 如果 Twitter 帐户是个人帐户，并且与事件或关联无关，则可能是过度共享。 只需要句柄： @elonmusk
5. **上下文角色：** 您可以在此控制您的事件宿主、演示者、审查人员以及事件内其他角色的超级能力或 [上下文角色](../world-building/granting-roles.md) 。 若要添加它们，请输入其 AltspaceVR 用户名，并在下拉菜单中为他们分配一个角色。 若要删除它们，请选中复选框以删除并保存事件网页。 注意：如果你想要授予演示者审核功能，则必须将其添加为演示者和审查方。 当前提供以下上下文角色：
    * **演示者：** 将 [前行功能](../faqs/front-row-events.md) 添加到 AltspaceVR 中的个人界面，以添加这些功能。
    * **审查方：** 向单独的界面添加了审核功能，包括能够为每个与会者提供文本、对事件进行全局静音或将其从事件中删除。 再次添加演示者，并向他们授予审核角色。
    * **仅扩音器：** 通过 "主机工具" 按钮将 "放大 My Voice" 按钮添加到其界面。
    * **Terraformer：** 将 "世界编辑器" 按钮添加到其接口。
    * **试点：** 为该个体添加航班功能。 注意：必须在 "设置/输入/动态" 中启用此项
    * **音乐执行者：** 添加与音乐会事件和乐谱事件关联的功能和功能。
6. **阻止列出的用户：** 如果你希望阻止用户访问，或以前由版主或表示器从事件中删除了用户，并且复制了事件，则将列出列出的用户的名称。 事件宿主或管理员可以随时添加或删除列出的块。

窗体完成并进行三次检查后，选择 " **创建事件**"。

> [!IMPORTANT]
> 如果未正确创建事件，请确保说明中至少有10个字符，并且/或者选择了模板，如果选中，模板名称将从白色变为蓝色。

## <a name="event-page-actions"></a>事件页操作

在事件网页上，你具有以下操作和选项：

1. 选择 " **编辑** " 更改事件设置。
2. 如果事件过早 (例如) ，请选择 " **结束事件** 结束事件"。
3. 选择 " **设置为草稿** " 将事件设置为 "草稿" 而不是 "活动"。
4. 如果你已对导入的世界进行更新，则需要选择 " **重新导入** "，以使这些更改出现在你的事件中，请不要忘记重置事件中的空间： ) 
5. 选择 " **重复事件** "，为将来的事件复制事件。 此重复项将出现在 "我的事件/我的草稿事件" 中，你将需要更新新的日期/时间，并从该处激活。
6. 选择 " **添加为主事件** " 可将事件设置到日历列表。
7. 如果要完全删除事件，请选择 " **删除事件** " (无法恢复此) 。

准备就绪后，请转到 AltspaceVR 的 "我的事件" 菜单中的 " **> 事件** "，并输入用于重置空间的事件空间、检查更改，或继续自定义。

## <a name="branding-your-event-with-images"></a>用图像标记事件

在 AltspaceVR 中，可以轻松地将图像添加到事件空间或公司的徽标和品牌，甚至您的团队或其他视觉对象的照片。

映像要求如下：
* 已接受 JPG 或 PNG 文件
* 最大图像大小：1920x1080
* 解决方法： 72 DPI
* 文件大小：小于每个文件 250 KB

此过程包括将图像上传到 [AltspaceVR 照片](https://account.altvr.com/photos)，然后使用世界上世界编辑器中的照片将其导入到事件世界空间。

1. 在 AltspaceVR 网站上，请参阅 " [照片](https://account.altvr.com/photos) "。
2. 选择“上传”。
3. 选择 " **浏览** " 以在计算机上打开一个对话框，然后查找并选择已优化的映像。
4. 选择“打开”。
5. 选择 " **创建照片**"。
6. 对所有映像重复此操作。

在 AltspaceVR 和事件世界中，将您自己放在要放置图像的位置附近。  (如果你要为 **重复使用事件** 创建此项，请确保你在世界而不是在事件空间中！ ) 

1. 在用户界面上，选择右下角的 **"世界编辑器/编辑器" 面板**
2. 单击底部的 " **我** 的" 选项卡。
3. 选择 " **照片**"。
4. 照片按上传的时间顺序存储，因此，最新上传的图像应该是列表中的第一个图像。 选择它以将其添加到全球。
5. 使用设备光标获取映像，并将其放置在所需的位置。
    * 您可以使用控制器调整其大小， (通常在 VR 的触摸板上横向滑动拇指，或在2D 模式下使用鼠标) 。
    * 若要微调图像的位置和旋转，请在编辑器中选择图像的 **齿轮** 符号，并适当地设置旋转和位置。
    * 还可以通过缩放功能调整图像大小。

> [!NOTE]
> 当编辑模式 (橙色文本) 上时，将自动具有动态功能，从而在事件世界中更轻松地放置图像。

6. 为每个图像位置重复此操作。
7. 设置映像后，选择 " **全部锁定** " （如果未逐个锁定这些映像），然后关闭编辑模式并关闭编辑器。
8. 每个事件最多只能使用五张照片。

## <a name="adding-3d-objects-using-world-editor-and-kits"></a>使用世界编辑器和工具包添加3D 对象

AltspaceVR 允许通过世界编辑器将3D 对象导入世界。 目前有很多世界编辑包可供使用，并且更多。 其中包括宇宙飞船、rockets、动画动物、sculptures、树、植物和多个不同的对象。 或者，你可以将自己的工具包添加到公共集合或个人用途，或考虑使用一个 [ALTSPACEVR MRE 应用](../world-building/using-mixed-reality-extensions.md)，该应用将种不可交互对象引入你的世界，如 helmets、双刃剑、wings、服装、帽子或其他种不可交互对象。 预计对象列表会很快展开。

此处的说明侧重于使用包中已有的项目。 有关在 AltspaceVR 中添加和导入三维对象的详细信息，请参阅 ["如何将 glTF 模型导入我的世界？"](../world-building/importing-models.md) 和 ["如何实现上传我自己的工具包？"](../world-building/uploading-custom-kits.md)。

若要将来自世界建筑套件的现有3D 对象添加到事件世界，请在 "世界编辑器/编辑器" 面板打开并启用编辑模式：

1. 浏览要导入的对象的工具包并将其选中。
2. 在工具包对象上选择光标，并将其定位。 使用 "世界" 编辑器中的对象上的齿轮图标来使用手动编号条目的 "位置"、"旋转" 和 "缩放" 选项。
3. 当对象置于正确的位置时，单击锁图标锁定它。
4. 如果需要，可添加另一个对象。 要明智地添加3D 对象，请将所做的选择保留为不超过五个不同工具包中的项目，以便获得更好的 Oculus 的用户支持。 不要在世界各地使用超过200个对象。
5. 完成后，锁定所有项，关闭编辑模式，然后关闭世界编辑器。 游离。 邀请几个测试人员进行访问和查看，并为您的出色活动做好准备！

若要深入了解全球编辑器工具包并向 AltspaceVR 添加3D 对象，请执行以下操作：

* [什么是早期访问计划？](../world-building/early-access.md)
* [如何将 glTF 模型导入我的世界？](../world-building/importing-models.md)
