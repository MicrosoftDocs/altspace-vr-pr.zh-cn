# <a name="creating-an-event"></a>创建活动

这是在 AltspaceVR 中创建事件的分步指南。 强烈建议你在 AltspaceVR 中出席几个事件，以了解其工作原理。 检查 [AltspaceVR 事件日历](https://account.altvr.com/events) 以获取所有事件的列表。

本文内容：

* AltspaceVR 事件术语
* 创建事件和事件空间或世界
* 使用用于幻灯片演示的多媒体控制台
* 用图像标记事件
* 使用全球编辑器/工具包添加3D 对象
* 如何记录或实时流我的事件

## <a name="altspacevr-event-terminology"></a>AltspaceVR 事件术语

以下是需要熟悉的术语来创建事件和事件空间：

</br>

| 术语 | 定义 |
|---|---|
| 世界建筑 | AltspaceVR 提供了创建和自定义虚拟世界的能力。 AltspaceVR 托管支持文档、Discord 通道和全球事件，可帮助你 [了解有关如何获取帮助和开始生成的详细信息](../world-building/world-building-faq.md)。 |
| World | 世界是 AltspaceVR 中的虚拟空间。 它可能是办公室或大量的山地范围。 它是高度可自定义的环境。 它位于一个范围内，在该范围内可能有多个领域。 如果希望有多个事件空间用于特殊事件、培训中心和不同的会议空间来情趣，请将这两个领域添加到同一个 Universe，使其成为一个组。 |
| Universe | AltspaceVR world 构建 vernacular 中的宇宙代表了世界的分类。 每个 universe 可以拥有多个领域。 世界继承了宇宙的设置，使用户可以轻松地将人员添加到多个领域的允许列表和其他控制功能。 有关详细信息，请参阅 [管理您的世界](../world-building/managing-worlds.md) 。 |
| 模板 | 模板 (或空间模板) 是可以使用的预建世界或环境，而不是使用世界建筑功能创建一个。 AltspaceVR 为不同的体验和事件提供各种模板。 |
| 事件空间 | 事件空间是 AltspaceVR 中世界的同义词。 通常，它是指用来托管事件的世界。 |
| 网站 | 对 "网站" 的引用是 [AltspaceVR 的网站](https://altvr.com/)。 通常通过计算机或平板电脑上的 [事件网页](https://account.altvr.com/events/my) 创建和编辑事件，而不是通过 VR 设备来创建和编辑事件。 还需要访问网站以进行广泛的 [构建](../world-building/managing-worlds.md) 。 |
| 上下文角色 | [上下文角色](../getting-started/roles.md) 由事件创建者或世界生成器分配。 这些角色为用户提供了世界级或事件空间额外的特性和功能。 目前，这包括主机、版主、试点 (航班) 、Terraformer (world 构建) 和扩音器。 可以单独或全局分配这些角色，使每个人都可以在事件空间或世界中拥有相同的角色。 |
| 用户界面 (UI) /Menu | 当你位于世界各地的 AltspaceVR 中时，屏幕左侧和右侧会出现菜单。 带有 AltspaceVR 徽标的圆圈或主菜单将打开 (UI) 或菜单上的主用户界面，以访问不同的屏幕来浏览 AltspaceVR 和自定义你的体验。 可选的 UI 项显示在屏幕的正确大小，通常包括世界编辑器和主机工具。 通过使用光标单击它们来打开和交互。 |
| SDK/MRE | 这些是与软件开发工具包和 [混合现实扩展](../world-building/using-mixed-reality-extensions.md) 相关联的世界建筑术语，用于向世界建筑体验添加特性和功能。 这通常适用于更高级的用户。 |

## <a name="understanding-events"></a>了解事件

使用 AltspaceVR 可以轻松地创建一个世界，你可以使用预置的模板空间根据特定需求使用或自定义，或者从头开始创建一个世界。 本文将介绍如何使用预置的模板来创建事件。 以下各节介绍 [了如何使用图像对事件进行标记](#branding-your-event-with-images) ，以及如何 [使用世界编辑器和工具包添加3d 对象](#adding-3d-objects-using-world-editor-and-kits) 。 若要深入了解如何从头开始构建自定义世界，请在 AltspaceVR 中参加聚会的建设教程，并 [查看全球构建支持文档](../world-building/world-editor-getting-started.md)。

AltspaceVR 提供了两种方法来为事件创建空间。

* **一次性使用：** 创建一个事件并选择一个模板 world。
* **重复使用：** 创建一个世界空间，并将其导入该事件。

对于这两种情况，创建事件的过程是相同的，但有一个例外： [创建一个世界，然后将其导入](../world-building/managing-worlds.md) 为重复使用的事件空间。 还需要了解：

</br>

| 术语 | 定义 |
|---|---|
| 世界复制 | 为重复事件使用世界空间允许持续自定义。 签名、图像、生成点和其他自定义项将从事件保留到事件，而不是每次都自定义事件空间。 |
| 自定义事件世界 | 创建事件时，会将模板或导入世界复制并锁定到该事件中。 您可以对事件世界进行更改，而不会影响原始世界以及其他方法。 请考虑添加节日装饰、图片或其他装饰一次性使用世界编辑器元素，使事件空间更愉快，并适合事件。 |

除非更新了事件空间，否则对原始环境所做的修改不会处于事件世界中：
1. 使用事件网页上的 " **重新导入" 世界** 按钮。
2. 等待完成同步所需的2-3 分钟。 
3. 如果事件空间尚未更改，请继续 **设置 > 中等 > 重置空间** 来重置事件空间。 你需要将空间设置为世界生成器！

如果遇到技术问题，如未正确加载元素，请在 "AltspaceVR" 菜单中选择 "**设置 > 适中 > 重置空间** 以重置事件空间并查看新的更改。 Windows2D PC 用户可以使用键盘快捷方式：在 AltspaceVR 中按 **CTRL + ALT + R** 可快速重置空间。

## <a name="creating-your-event-and-event-space-or-world"></a>创建事件和事件空间或世界

下面是有关为一次性事件或重复事件创建事件的分步说明。 选择模板或导入事件的世界。 

> [!NOTE]
> AltspaceVR 事件网页通过使用绿色问号按钮，提供窗体的每个方面的说明。 将光标滚动到它们上以获得特定说明。

在 AltspaceVR 网站上的 " [事件" > 我的事件](https://account.altvr.com/events/my) "网页上，选择" **计划事件** "或直接转到" AltspaceVR "网页中的" [创建事件](https://account.altvr.com/events/new)"。

1. **事件标题：** 键入事件的名称。 使其在 AltspaceVR 的网站和全球界面的各种日历视图中保持特定和简明。 尝试将标题保留为23个字符（包括单词之间的空格）。
2. **说明：** 键入事件的描述。
    * 说明中必须至少有10个字符，否则不会创建事件。
    * 在段落之间增加空白间距。
    * 若要添加指向事件 Facebook、Discord、网站或其他资源的链接，请使用以下格式 (此 Markdown 仅适用于网站上的事件升级页面，这在 AltspaceVR 菜单中无法正确呈现) ： `[Event Name](http://example.com/)`

3. **开始日期/结束日期：** 设置开始时间并确保结束时间晚到开始时间。
4. **类别：** 选择最能描述你的事件类型的类别。 
5. 将事件设置为 " **专用** " 或 " **公共**"。
    * 可见性：公共事件显示在 AltspaceVR 网站事件日历的 " [所有" 选项卡](https://account.altvr.com/events/all) 上，并公开到 "公共"。
    * 私有事件不显示在 AltspaceVR 事件日历上，并且需要事件 URL 来输入事件空间。
    * 如果希望 "添加为主事件"，则必须将事件设置为 "公共"。
6. **选择模板：** 网页右侧是 AltspaceVR 中可用模板的缩略图图像列表。 有游戏房间、办公室、会议室、演示文稿空间和有趣的会议空间。 选择一个看起来有趣的选项。 如果不喜欢，请继续操作，只需使用新模板创建新事件。 如果要创建自己的自定义事件世界，请选择 **"Cool Sky"** 作为默认模板，然后按照以下说明导入你的世界。
7. 选择 **"高级选项"**

### <a name="advanced-options"></a>高级选项

1. **提升：** 每个事件都需要两个品牌图像 (这些图像显示在 AltspaceVR 网站上，并且不会显示在 AltspaceVR) ：
    * **磁贴：** 磁贴图像需要为 1920x1080 像素。 此图像将缩放为各种缩略图大小，并与其他事件一起显示在 AltspaceVR 事件日历上。 确保图像清晰，并且其上没有太多文本。 请勿使用自己没有的受版权保护的图像/徽标。
    * **横幅：** 横幅图像需要为 1920x576。 这将根据需要自动调整大小，在事件信息或网页上可见。 半透明覆盖覆盖图像中较低的 25%。 避免将文本置于该区域中。
    * 另 (一) 选项是"将磁贴图像复制到 **背景** 横幅图像"，这也将使用磁贴图像作为横幅图像。 试一试 ：) 。 如果外观不良好，可以创建一个新的横幅图像。
2. **在 VR 中：** 本部分包括适用于 AltspaceVR 应用内事件期间虚拟体验的功能：
    * **默认上下文角色：** 绿色问号列出了事件 中所有受众 *成员可用的特定角色*。 最常见的是 *megaphone_only。* 添加此选项以向所有受众成员提供"主机工具"按钮下的"放大我的语音"选项，这样，如果空间很大，可能会在整个活动中听到他们的声音。 如果事件需要航班，请添加 *试点*。 若要同时添加它们，格式为 *：megaphone_only，试点*。 每个用户需要在 AltspaceVR 应用中启用 Fly 模式，设置/输入/Fly。
    * **说明：** 说明 [适用于](../world-building/adding-welcome-messages.md) 在到达空间时生成欢迎图像的文本。 用户必须选择"确定"，以将其从视图中删除。 这通常用于提供受众说明，例如"请保持静音，直到受邀说话"或"欢迎参加 XYZ 活动，我们将在其中涵盖 ABC"。 这不是必需的，因此请特意使用 。 还可使用 Markdown 添加颜色和字体大小调整，更多详细信息： [http://digitalnativestudios.com/textmeshpro/docs/rich-text](http://digitalnativestudios.com/textmeshpro/docs/rich-text)
3. **高级：** 以下是事件的高级功能的快速说明 (其中一些功能在事件创建后才会显示，并且编辑事件) ：
    * **管理员：** 管理员是信任的 AltspaceVR 用户，可帮助你管理事件。 它们可能是你的共同主机或备份主机。 通过将用户名添加到管理员列表，他们可以：
        * 更改事件标题、说明和其他功能。
        * 添加和删除审查器、主机和其他角色的上下文角色。
        * 删除事件。
    * **组：** 使用下拉菜单 [从](group-features.md) 专用组进行选择。 仅在帐户已创建/已添加到组时显示。
    * **标记：** 此简洁的句子将显示在事件网页的标题下。
    * **从世界导入：** 若要使用已创建世界事件空间重复使用，这是窗体的 一部分，用于将世界布局和自定义导入事件。 请注意以下注意事项：

    **导入自己的世界：** 若要导入世界，请创建并拥有：
    * 选择下拉箭头以查看已创建世界的列表。
    * 选择世界。
    * 完成事件窗体的其余部分后，请等待至少 2 分钟，然后访问 AltspaceVR 中的事件空间，以确保更新数据库信息并生成世界。
    
    **使用"他人世界"：**
    * 联系世界所有者，请求他们使用个人世界 **上的** "与好友共享"功能与你共享。

3. **YouTube 视频 ID：** 在活动的网页上可见，这会将 YouTube 事件尾部或视频添加到事件网页，而不是向全球添加。 只需 URL 的视频 ID 部分：dQw4w9WgXcQ
4. **Twitter 句柄：** 这会将 Twitter 流添加到事件的网页。 如果 Twitter 帐户是个人帐户，与事件或关联不相关，则可能是过度共享。 你只需句柄： @elonmusk
5. **上下文角色：** 在这里，你可以控制事件主机、审查器 [](../world-building/granting-roles.md)以及事件内其他角色的超级权限或上下文角色。 若要添加它们，请在 AltspaceVR 用户名中输入 ，然后从下拉菜单中为其分配角色。 若要删除它们，请选中"删除并保存事件网页"复选框。 注意：如果要授予主机审查功能，必须将它们添加为主机和审查器。 目前提供以下上下文角色：
    * **主机：** 将 [Front Row 功能](../faqs/front-row-events.md) 添加到 AltspaceVR 中的个人界面，以添加这些功能。
    * **审查器：** 向个人界面添加审核功能，包括向每个参与者发送文本、为活动全局静音或删除活动的功能。 再次添加主机，并授予其审核角色（如果希望他们具有审核权限）。
    * **仅兆音电话：** 通过"主机工具"按钮将"放大我的语音"按钮添加到其界面。
    * **Terraformer：** 将"世界编辑器"按钮添加到其界面。
    * **试点：** 为该个人添加航班功能。 注意：他们必须在 设置/Input/Fly 中启用此功能
    * **音乐执行者：** 添加与音乐会和音乐活动关联的功能和特征。
6. **阻止列出的用户：** 如果要阻止用户访问，或者用户以前已被审查器或主机从事件中删除，并且事件重复，则列出的阻止用户名将列出。 事件主机或管理员可以随时添加或删除列出的阻止用户。

完成窗体并选中三次后，选择"**创建事件"。**

> [!IMPORTANT]
> 如果事件未正确创建，请确保"说明"中至少有 10 个字符和/或已选择"模板"，如果选择了"模板"，则模板的名称将从白色变成蓝色。

2. 选择 **"结束** 事件"以结束事件 (例如，如果事件提前结束，) 。
3. 选择 **"设置为草稿** "，将事件设置为"草稿"而不是"活动"。
4. 如果已对导入世界进行了更新，则需要选择" **重新** 导入世界"，使这些更改出现在事件中，不要忘记在 事件 ：) 
5. 选择 **"复制** 事件"，复制事件供将来事件使用。 然后，此重复项将显示在"我的事件/我的草稿事件"中，你需要更新新的一天/时间，然后从该时间激活它。
6. 选择 **"添加 AS MAIN 事件** "，将事件设置为日历列表。
7. 如果要 **完全** 删除事件，请选择"删除事件 (无法恢复此) 。

## <a name="branding-your-event-with-images"></a>使用图像为事件设置品牌

在 AltspaceVR 中，将图像添加到活动空间或公司的徽标和品牌，甚至是你自己、团队或其他视觉内容的照片。

映像要求包括：
* 接受 JPG 或 PNG 文件
* 最大图像大小：1920x1080
* 分辨率：72 DPI
* 文件大小：每个文件小于 250 KB

此过程涉及将图像上传到 [AltspaceVR 照片](https://account.altvr.com/photos)，然后使用"世界编辑器"中的"世界照片"将其导入并放入事件世界空间。

1. 转到[](https://account.altvr.com/photos)AltspaceVR 网站上的照片。
2. 选择“上传”。
3. 选择 **"** 浏览"，在计算机上打开一个对话框，找到并选择优化的映像。
4. 选择“打开”  。
5. 选择"**创建照片"。**
6. 针对所有映像重复此操作。

在 AltspaceVR 和事件世界中，将你自己定位在要放置图像的位置附近。  (如果要为重复使用事件创建此 ，请确保你在你的世界中，而不是在事件空间中！) 

1. 在用户界面的右下角，选择"世界 **编辑器/编辑器面板"**
2. 单击底部的 **"挖掘** "选项卡。
3. 选择"**照片"。**
4. 照片按上传时间顺序存储，因此上传的最新图像应为列表中的第一个图像。 选择它以将其添加到世界。
5. 使用设备光标抓取图像，将图像放置在你希望的位置。
    * 可以使用控制器调整其大小， (在 VR 中的触摸板上水平滑动滚动块，或在 2D 模式下使用鼠标) 。
    * 若要微调图像的位置和旋转，请在编辑器中选择图像上的 **"齿轮** "符号，并适当地设置"旋转"和"位置"。
    * 还可使用缩放功能调整图像的大小。

> [!NOTE]
> 当"编辑模式" (橙色文本) ，你将自动能够进行飞行，从而更轻松地移动图像，从而在事件世界中将图像放置在更高的水平。

6. 对每个图像放置重复此操作。
7. 设置图像后，如果未单独锁定图像，请选择"全部锁定"，然后关闭"编辑模式"并关闭编辑器。
8. 每个事件最多只能使用五张照片。

## <a name="adding-3d-objects-using-world-editor-and-kits"></a>使用世界编辑器和工具包添加 3D 对象

AltspaceVR 允许使用世界编辑器将 3D 对象导入世界。 目前有许多世界编辑器工具包可供使用，并且更多编辑器工具包正在使用中。 有太空飞船、火箭、动画动物、鸟、树、植物和许多不同的对象。 或者，可以将自己的工具包添加到公共集合或供自己的个人使用，或者考虑使用 [AltspaceVR MRE](../world-building/using-mixed-reality-extensions.md)应用之一，将可交互的对象（如、鸟、、服装、帽或其他可交互对象）引入你的世界。 预计对象列表很快会展开。

此处的说明侧重于使用工具包中已有的项。 有关在 AltspaceVR 中添加和导入 3D 对象的信息，请参阅 ["如何将 glTF 模型导入我的世界？"](../world-building/importing-models.md) 和["如何实现上传自己的工具包？"。](../world-building/uploading-custom-kits.md)

若要将世界建筑工具包中的现有 3D 对象添加到事件世界，请打开"世界编辑器/编辑器面板"并启用"编辑模式"：

1. 浏览要导入的对象的工具包并选择它。
2. 使用光标选择工具包对象并定位它。 使用"世界编辑器"中对象上的"齿轮"图标，对手动数字输入使用"位置"、"旋转"和"缩放"选项。
3. 将对象置于正确的位置时，单击"锁定"图标将其锁定。
4. 如果需要，请添加另一个对象。 在添加 3D 对象时要十分明智，将所选内容保留为不超过五个不同单独工具包中的项，以更好地获得 Oculus Quest 用户支持。 请勿在一个世界内使用超过 200 个对象。
5. 完成后，锁定所有项，关闭"编辑模式"，然后关闭"世界编辑器"。 四处。 邀请一些测试人员来访问并查看，并准备好参加你的活动！
