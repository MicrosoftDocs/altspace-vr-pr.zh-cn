# <a name="creating-an-event"></a>创建活动

这是在 AltspaceVR 中创建事件的分步指南。 强烈建议你参加 AltspaceVR 中的多个活动，了解其工作状态。 查看 [AltspaceVR 事件日历](https://account.altvr.com/events) ，查看所有事件的列表。

本文内容：

* AltspaceVR 事件术语
* 创建事件和事件空间或世界
* 使用多媒体控制台进行幻灯片演示
* 使用图像为事件设置品牌
* 使用世界编辑器/工具包添加 3D 对象
* 如何记录或实时流我的事件

## <a name="altspacevr-event-terminology"></a>AltspaceVR 事件术语

以下是创建事件和事件空间时需要熟悉的术语：

</br>

| 术语 | 定义 |
|---|---|
| 世界建筑 | AltspaceVR 提供创建和自定义虚拟世界的能力。 AltspaceVR 主机支持文档、分流通道和世界内活动，帮助你详细了解如何 [获取帮助并开始构建世界](../world-building/world-building-faq.md)。 |
| World | 世界是 AltspaceVR 中的虚拟空间。 它可能是办公室或宽广的的范围。 它是高度可自定义的环境。 它位于一个世界内，并且该世界内可能有多个世界。 如果希望为特殊活动、训练中心和不同的会议空间创建多个活动空间来使事件变得十分丰富，请在同一个 Universe 下添加世界，将它们作为一个组一起保存。 |
| 宇宙 | AltspaceVR 世界建筑中的一个世界代表世界分类。 每个世界可能包含多个世界。 世界继承了世界的设置，因此可以轻松将人员添加到多个世界和其他控制功能的允许列表。 有关详细信息 [，请参阅](../world-building/managing-worlds.md) 管理你的世界。 |
| 模板 | 模板 (或空间模板) 是一个预构建世界或环境，可以代替使用世界建筑功能创建一个。 AltspaceVR 为不同的体验和事件提供各种模板。 |
| 事件空间 | 事件空间是 AltspaceVR 中的世界同义词。 通常，它是指用于托管事件的世界。 |
| 网站 | 对"网站"的引用是 [AltspaceVR 的网站](https://altvr.com/)。 通常，通过计算机或平板电脑上的"事件"网页创建和编辑[](https://account.altvr.com/events/my)事件比通过 VR 设备创建和编辑事件更容易。 你还需要访问用于世界 [建筑](../world-building/managing-worlds.md) 的网站。 |
| 上下文角色 | [上下文角色](../getting-started/roles.md) 由事件创建者或世界生成器分配。 这些角色为世界或事件空间中的用户提供额外的特性和功能。 目前，其中包括主机、审查器、 () 、Terraformer (world building) 和 Megaphone。 这些角色可以单独或全局分配，允许每个人在事件空间或世界中具有相同的角色。 |
| 用户界面 (UI) /Menu | 在沉浸式环境中，当你在 AltspaceVR 中时，屏幕左侧和右侧有菜单。 使用 AltspaceVR 徽标的圆圈或主菜单将打开 UI (或菜单) 以访问不同的屏幕，以浏览 AltspaceVR 和自定义体验。 可选 UI 项位于屏幕的合适大小上，通常包括"世界编辑器"和"主机工具"。 通过用光标单击它们来打开它们并与之交互。 |
| SDK/MRE | 这些是世界性构建术语，与软件开发工具包和混合现实扩展[](../world-building/using-mixed-reality-extensions.md)相关联，用于向世界生成体验添加特性和功能。 这些通常适用于更高级的用户。 |

## <a name="understanding-events"></a>了解事件

使用 AltspaceVR 可以轻松创建一个可以使用预先创建的模板空间来根据特定需求使用或自定义世界，或者从头开始创建一个世界。 本文介绍使用预先创建的模板创建事件。 以下有关使用图像打造事件 [品牌](#branding-your-event-with-images) 和使用世界编辑器和工具包添加 [3D](#adding-3d-objects-using-world-editor-and-kits) 对象的部分包含有关进一步自定义的提示。 有关从头开始构建自定义世界的信息，请参加 AltspaceVR 中的世界建筑教程会议，并查看世界建筑 [支持文档](../world-building/world-editor-getting-started.md)。

AltspaceVR 提供了两种为事件创建空间的方法。

* **一次使用：** 创建事件并选择模板世界。
* **重复使用：** 创建世界空间，将其导入事件。

创建事件的过程对于两者都是相同的，有一个例外：创建一个世界，并导入 [它作为重复](../world-building/managing-worlds.md) 使用的事件空间。 还需要了解：

</br>

| 术语 | 定义 |
|---|---|
| 已复制世界 | 对重复事件使用世界空间允许持久自定义。 在事件之间维护符号、图像、生成点和其他自定义项，而不是每次自定义事件空间。 |
| 自定义事件世界 | 创建事件时，模板或导入世界将被复制并锁定到该事件中。 你可以对事件世界进行更改，而不会影响原始世界，而不会影响另一方面。 请考虑添加假日装饰、图片或其他装饰性一次使用世界编辑器元素，使事件空间更可玩且更适合于活动。 |

除非更新事件空间，否则对原始世界所做的修改不会在事件世界内：
1. 使用 **事件网页上的"重新** 导入世界"按钮。
2. 等待 2-3 分钟，同步完成。 
3. 如果事件空间未更改，请转到"设置">"> **重置** 事件空间"。 作为世界构建者，你将重置大量空间！

如果存在技术问题（例如元素未正确加载），请转到 AltspaceVR 菜单，然后选择"设置">"中等> **重置空间"** 以重置事件空间并查看新更改。 2D 中的 Windows 电脑用户可以使用键盘快捷方式 **：Ctrl + Alt + R** 在 AltspaceVR 中快速重置空间。

## <a name="creating-your-event-and-event-space-or-world"></a>创建事件和事件空间或世界

以下是为一次或重复事件创建事件的分步说明。 通过选择模板或导入事件世界。 

> [!NOTE]
> AltspaceVR 事件网页使用绿色问号按钮，提供表单的各个方面的说明。 将光标滚动到它们上，以使用特定说明。

在[](https://account.altvr.com/events/my)AltspaceVR >"我的事件"网页的"事件"页上，选择"计划事件"或直接转到["在 AltspaceVR 中创建事件"网页](https://account.altvr.com/events/new)。

1. **事件标题：** 键入事件的名称。 保持具体简洁，以便查看 AltspaceVR 网站和世界界面中的各种日历视图。 尝试将标题保留为 23 个字符（包括单词之间的空格）。
2. **说明：** 键入事件的说明。
    * 说明中必须至少有 10 个字符，否则不会创建事件。
    * 在段落之间添加空白区域。
    * 若要添加指向活动的 Facebook、分步器、网站或其他资源的链接，请使用以下格式 (此 Markdown 仅适用于网站的"活动推广"页，这不会在 AltspaceVR 菜单菜单中正确呈现) ： `[Event Name](http://example.com/)`

3. **开始日期/结束日期：** 设置开始时间，并确保结束时间在开始时间之后。
4. **类别：** 选择最能描述事件类型的类别。 
5. 将事件设置为 Private **或** **Public**。
    * 可见性：公共事件在 AltspaceVR 网站事件日历的"所有"[](https://account.altvr.com/events/all)选项卡上可见，并公开。
    * 专用事件在 AltspaceVR 事件日历上不可见，需要事件 URL 才能输入事件空间。
    * 如果要"添加为主事件"，则必须将事件设置为 public。
6. **选择模板：** 网页右侧是 AltspaceVR 中可用模板的缩略图图像列表。 有游戏房间、办公室、会议室、演示文稿空间和有趣的会议空间。 选择一个看起来有趣的选项。 如果不喜欢，请继续操作，只需使用新模板创建新事件。 如果要创建自己的自定义事件世界，请选择 **"Cool Sky"** 作为默认模板，然后按照以下说明导入你的世界。
7. 选择 **"高级选项"**

### <a name="advanced-options"></a>高级选项

1. **提升：** 每个事件都需要两个品牌图像 (这些图像显示在 AltspaceVR 网站上，并且不会显示在 AltspaceVR) ：
    * **磁贴：** 磁贴图像需要为 1920x1080 像素。 此图像将缩放为各种缩略图大小，并与其他事件一起显示在 AltspaceVR 事件日历上。 确保图像清晰，并且其上没有太多文本。 请勿使用自己没有的受版权保护的图像/徽标。
    * **横幅：** 横幅图像需要为 1920x576。 这将根据需要自动调整大小，在事件信息或网页上可见。 半透明覆盖覆盖图像中较低的 25%。 避免将文本放置在该区域中。
    * 另 (一) 选项是"将磁贴图像复制到 **背景** 横幅图像"，这也将使用磁贴图像作为横幅图像。 试一试 ：) 。 如果外观不良好，可以创建一个新的横幅图像。
2. **在 VR 中：** 本部分包括适用于 AltspaceVR 应用内事件期间虚拟体验的功能：
    * **默认上下文角色：** 绿色问号列出了事件 中所有受众 *成员可用的特定角色*。 最常见的是 *megaphone_only。* 添加此选项以向所有受众成员提供"主机工具"按钮下的"放大我的语音"选项，这样，如果空间很大，可能会在整个活动中听到他们的声音。 如果事件需要航班，请添加 *试点*。 若要同时添加它们，格式为 *：megaphone_only，试点*。 每个用户都需要通过进入设置/输入/Fly 在 AltspaceVR 应用中启用 Fly 模式。
    * **说明：** 说明 [适用于](../world-building/adding-welcome-messages.md) 在到达空间时生成欢迎图像的文本。 用户必须选择"确定"，以将其从视图中删除。 这通常用于提供受众说明，例如"请保持静音，直到受邀说话"或"欢迎参加 XYZ 活动，我们将在其中涵盖 ABC"。 这不是必需的，因此请特意使用 。 还可使用 Markdown 添加颜色和字体大小调整，更多详细信息： [http://digitalnativestudios.com/textmeshpro/docs/rich-text](http://digitalnativestudios.com/textmeshpro/docs/rich-text)
3. **高级：** 以下是事件的高级功能的快速说明 (其中一些功能在事件创建后才会显示，并且编辑事件) ：
    * **管理员：** 管理员是信任的 AltspaceVR 用户，可帮助你管理事件。 它们可能是你的共同主机或备份主机。 通过将用户名添加到管理员列表，他们可以：
        * 更改事件标题、说明和其他功能。
        * 添加和删除审查器、主机和其他角色的上下文角色。
        * 删除事件。
    * **组：** 使用下拉菜单 [从](group-features.md) 专用组进行选择。 仅在帐户已创建/已添加到组时显示。
    * **标记：** 此简洁的句子将显示在事件网页的标题下。
    * **从世界导入：** 若要使用创建世界事件空间重复使用，这是窗体的 一部分，用于将世界布局和自定义导入事件。 请注意以下注意事项：

    **导入自己的世界：** 若要导入世界，请创建并拥有：
    * 选择下拉箭头以查看已创建世界的列表。
    * 选择世界。
    * 完成事件窗体的其余部分后，请等待至少 2 分钟，然后访问 AltspaceVR 中的事件空间，以确保更新数据库信息并生成世界。
    
    **使用"他人世界"：**
    * 联系世界所有者，请求他们使用个人世界 **上的** "与好友共享"功能与你共享。

3. **YouTube 视频 ID：** 在活动的网页上可见，这会将 YouTube 事件尾部或视频添加到事件网页，而不是向全球添加。 只需 URL 的视频 ID 部分：dQw4w9WgXcQ
4. **Twitter 句柄：** 这会将 Twitter 流添加到事件的网页。 如果 Twitter 帐户是个人帐户，与事件或关联不相关，则可能是过度共享。 你只需句柄： @elonmusk
5. **上下文角色：** 在这里，你可以控制事件主机、审查器 [](../world-building/granting-roles.md)以及事件内其他角色的超级权限或上下文角色。 若要添加它们，请在 AltspaceVR 用户名中输入 ，然后从下拉菜单中为其分配角色。 若要删除它们，请选中"删除并保存事件网页"复选框。 注意：如果要授予主机审查功能，必须将它们添加为主机和审查器。 目前提供以下上下文角色：
    * **主机：** 将 [Front Row 功能](../faqs/front-row-events.md) 添加到 AltspaceVR 中的个人界面，以添加这些功能。
    * **审查器：** 向个人界面添加审核功能，包括向每个参与者发送文本、为活动全局静音，或者从事件中删除这些功能。 再次添加主机，并授予其审核角色（如果希望他们具有审核权限）。
    * **仅兆音电话：** 通过"主机工具"按钮将"放大我的语音"按钮添加到其界面。
    * **Terraformer：** 将"世界编辑器"按钮添加到其界面。
    * **试点：** 为该个人添加航班功能。 注意：他们必须在"设置/输入/飞行"中启用此功能
    * **音乐执行者：** 添加与音乐会和音乐活动关联的功能和功能。
6. **阻止列出的用户：** 如果要阻止用户访问，或者用户以前已被审查器或主机从事件中删除，并且事件重复，则列出的阻止用户名将列出。 事件主机或管理员可以随时添加或删除列出的阻止用户。

完成窗体并选中三次后，选择"**创建事件"。**

> [!IMPORTANT]
> 如果事件未正确创建，请确保"说明"中至少有 10 个字符和/或已选择"模板"，如果选择了"模板"，则模板的名称将从白色变成蓝色。

## <a name="branding-your-event-with-images"></a>使用图像为事件设置品牌

在 AltspaceVR 中，将图像添加到事件空间或公司的徽标和品牌，甚至包括你自己、团队或其他视觉内容的照片。

映像要求包括：
* 接受 JPG 或 PNG 文件
* 最大图像大小：1920x1080
* 解决方法： 72 DPI
* 文件大小：小于每个文件 250 KB

此过程包括将图像上传到 [AltspaceVR 照片](https://account.altvr.com/photos)，然后使用世界上世界编辑器中的照片将其导入到事件世界空间。

1. 在 AltspaceVR 网站上，请参阅 " [照片](https://account.altvr.com/photos) "。
2. 选择“上传”。
3. 选择 " **浏览** " 以在计算机上打开一个对话框，然后查找并选择已优化的映像。
4. 选择“打开”  。
5. 选择 " **创建照片**"。
6. 对所有映像重复此操作。

在 AltspaceVR 和事件世界中，将您自己放在要放置图像的位置附近。  (如果你要为 **重复使用事件** 创建此项，请确保你在世界而不是在事件空间中！ ) 

1. 在用户界面上，选择右下角的 **"世界编辑器/编辑器" 面板**
2. 单击底部的 " **我** 的" 选项卡。
3. 选择 " **照片**"。
4. 照片按上传的时间顺序存储，因此，最新上传的图像应该是列表中的第一个图像。 选择它以将其添加到全球。
5. 使用设备光标获取映像，并将其放置在所需的位置。
    * 您可以使用控制器调整其大小， (通常在 VR 的触摸板上横向滑动拇指，或在2D 模式下使用鼠标) 。
    * 若要微调图像的位置和旋转，请在编辑器中选择图像的 **齿轮** 符号，并适当地设置旋转和位置。
    * 还可以通过缩放功能调整图像大小。

> [!NOTE]
> 当编辑模式 (橙色文本) 上时，将自动具有动态功能，从而在事件世界中更轻松地放置图像。

6. 为每个图像位置重复此操作。
7. 设置映像后，选择 " **全部锁定** " （如果未逐个锁定这些映像），然后关闭编辑模式并关闭编辑器。
8. 每个事件最多只能使用五张照片。

## <a name="adding-3d-objects-using-world-editor-and-kits"></a>使用世界编辑器和工具包添加3D 对象

AltspaceVR 允许通过世界编辑器将3D 对象导入世界。 目前有很多世界编辑包可供使用，并且更多。 其中包括宇宙飞船、rockets、动画动物、sculptures、树、植物和多个不同的对象。 或者，你可以将自己的工具包添加到公共集合或个人用途，或考虑使用一个 [ALTSPACEVR MRE 应用](../world-building/using-mixed-reality-extensions.md)，该应用将种不可交互对象引入你的世界，如 helmets、双刃剑、wings、服装、帽子或其他种不可交互对象。 预计对象列表会很快展开。

此处的说明侧重于使用包中已有的项目。 有关在 AltspaceVR 中添加和导入三维对象的详细信息，请参阅 ["如何将 glTF 模型导入我的世界？"](../world-building/importing-models.md) 和 ["如何实现上传我自己的工具包？"](../world-building/uploading-custom-kits.md)。

若要将来自世界建筑套件的现有3D 对象添加到事件世界，请在 "世界编辑器/编辑器" 面板打开并启用编辑模式：

1. 浏览要导入的对象的工具包并将其选中。
2. 在工具包对象上选择光标，并将其定位。 使用 "世界" 编辑器中的对象上的齿轮图标来使用手动编号条目的 "位置"、"旋转" 和 "缩放" 选项。
3. 当对象置于正确的位置时，单击锁图标锁定它。
4. 如果需要，可添加另一个对象。 要明智地添加3D 对象，请将所做的选择保留为不超过五个不同工具包中的项目，以便获得更好的 Oculus 的用户支持。 不要在世界各地使用超过200个对象。
5. 完成后，锁定所有项，关闭编辑模式，然后关闭世界编辑器。 游离。 邀请几个测试人员进行访问和查看，并为您的出色活动做好准备！

若要深入了解全球编辑器工具包并向 AltspaceVR 添加3D 对象，请执行以下操作：

* [什么是早期访问计划？](../world-building/early-access.md)
* [如何将 glTF 模型导入我的世界？](../world-building/importing-models.md)
