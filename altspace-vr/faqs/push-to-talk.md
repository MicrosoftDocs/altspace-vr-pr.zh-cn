# <a name="is-there-a-push-to-talk-button"></a>有一键通按钮吗？

在视图的右下角，有一个麦克风按钮可用于切换语音是否正在与其他用户共享。 或者，可以使用键盘快捷方式 **Ctrl + Alt + T** 来切换此设置。 
 
如果在说话时图标闪烁，则麦克风将正常工作！
