# <a name="is-there-a-push-to-talk-button"></a>是否有 "按下对话" 按钮？

在视图的右下角有一个麦克风按钮，可用于切换您的语音是否与其他用户共享。 或者，您可以使用键盘快捷方式 **Ctrl + Alt + T** 来切换此设置。 
 
如果说话时图标闪烁，则麦克风工作正常！
