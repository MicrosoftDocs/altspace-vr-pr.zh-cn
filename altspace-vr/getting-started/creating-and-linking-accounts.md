# <a name="creating-and-linking-your-altspacevr-account"></a>创建和链接 AltspaceVR 帐户

## <a name="how-to-create-an-account-in-vr"></a>如何在 VR 中创建帐户

使用 Microsoft 登录名创建帐户：

1. 首先，从 VR 应用商店或库下载 AltspaceVR 实时生成。 <br>*如果已有应用程序，请确保已安装最新更新。*
2. 启动应用程序并导航到左侧的"登录"按钮。
3. 选择"登录"按钮，然后选择"Microsoft 登录"按钮。
4. 将显示一个单独的屏幕，提示你通过需要复制的唯一代码激活设备。
6. 切换到计算机的浏览器屏幕并转到 https://www.microsoft.com/devicelogin
7. 输入唯一代码，然后选择"下一步"。 *代码不区分大小写。*
8. 按照屏幕上的说明激活设备。
9. 完成帐户链接过程后，系统会提示你填写显示名称和用户名。
10. 选择"注册"。

## <a name="how-to-create-an-account-in-2d-mode"></a>如何在 2D 模式下创建帐户

使用 Microsoft 登录名创建帐户：

1. 首先 [，从首选应用商店或库下载 AltspaceVR](https://altvr.com/getaltspacevr) 实时生成。 <br>*如果已有应用程序，请确保已安装最新更新。*
2. 启动应用程序。
3. 选择"登录"按钮，然后选择"Microsoft 登录"按钮。
3. 将打开一个单独的窗口，提示你选择想要使用的 Microsoft 帐户。 选择个人 Microsoft 帐户，而不是与工作或学校帐户关联的帐户。
4. 完成帐户链接过程后，系统会提示你填写显示名称和用户名。
5. 选择"注册"。

## <a name="linking-your-microsoft-account"></a>链接 Microsoft 帐户

可以从 AltspaceVR 应用程序内部链接预先存在的 AltspaceVR 帐户。

在电脑上与 2D 模式链接：

1. 确保已安装最新更新并启动 AltspaceVR。
2. 启动 AltspaceVR，并确保已使用经典 AltspaceVR 帐户登录。 如果在启动 AltspaceVR 时系统提示登录，请选择"登录"按钮，然后选择"使用 AltspaceVR 帐户登录"链接。
2. 在"标题屏幕"中，选择右侧"帐户"选项卡。 如果帐户未链接，则帐户状态应为"经典 Altspace 帐户"
3. 选择"链接 Microsoft 帐户"。 验证是否显示了正确的用户名，然后选择"链接"按钮。
4. 将打开一个单独的窗口，提示你选择要链接的 Microsoft 帐户。 选择个人 Microsoft 帐户，而不是与工作或学校帐户关联的帐户。
5. 按照链接说明操作。 完成帐户链接过程后，你将在菜单的"帐户"部分看到帐户凭据已更新。 状态应为"Microsoft 家庭帐户 (MSA) "

在头戴显示设备上与 VR 模式链接：

1. 确保已安装最新更新并启动 AltspaceVR。
2. 启动 AltspaceVR，并确保已使用经典 AltspaceVR 帐户登录。 如果在启动 AltspaceVR 时系统提示登录，请选择"登录"按钮，然后选择"使用 AltspaceVR 帐户登录"链接。
2. 在"标题屏幕"中，选择右侧"帐户"选项卡。 如果帐户未链接，则帐户状态应为"经典 Altspace 帐户"
4. 选择"链接 Microsoft 帐户"。 验证是否显示了正确的用户名，然后选择"链接"按钮。
4. 将显示一个单独的屏幕，提示你通过需要复制的唯一代码激活设备。
5. 切换到计算机的浏览器屏幕并转到 https://www.microsoft.com/devicelogin
6. 输入唯一代码，然后选择"下一步"。 *注意：代码不区分大小写。*
7. 按照屏幕上的说明激活设备。
8. 导航回 AltspaceVR 后。 在菜单的"帐户"部分中，应会看到帐户凭据已更新。 状态应为"Microsoft 家庭帐户 (MSA) "。

疑难解答提示:

* 如果在帐户链接过程中遇到任何问题，请尝试退出并关闭 AltspaceVR，然后重新尝试连接帐户。
* 如果进入 VR 模式并遇到问题，请尝试拔下头戴显示设备并切换到 2D 电脑模式 (或以其他方式) 。
* 如果发现自己无法重新连接Microsoft 帐户，仍可以使用用户名和密码返回到 AltspaceVR。 只需选择"使用 AltspaceVR 登录"按钮。
* 如果使用的是 VR，当系统提示打开对话框时，请尝试在已使用要与 AltspaceVR 一起使用的 Microsoft 凭据的电脑上执行此操作。
* 如果遇到任何问题，请通过"主菜单 **">"设置">">日志"提交日志**。

## <a name="account-registration-faq"></a>帐户注册常见问题解答

**其他用户将看到哪个名称？**

你的"显示名称"是其他用户将在头像上方看到的名称。 你的"用户名"将显示在菜单的"好友"选项卡中，供已添加你为好友的人使用。 "显示名称"和"用户名"都显示在头像的名称标签上。

**显示名称是否必须唯一？**
 
否，你的用户名可以与其他用户名共享相同的显示名称。 如果希望将显示名称编写为"Dave"或"Maria"，可以！ 显示名称还可以包括用户名通常不能包含的空格。

**我记不起来密码，如何重置密码？**

- 如果使用的是密码Microsoft 帐户，可以在启动登录过程后使用"Microsoft 登录"按钮执行 Microsoft 密码重置流。 
- 如果使用的是经典 AltspaceVR 帐户，可以使用"忘记了密码"链接重置密码，也可以在此处重置 [密码](https://account.altvr.com/users/password/new)。

**如何更改用户名或显示名称？**

如果要更改与帐户关联的用户名， [请向我们发送票证](https://help.altvr.com/hc/requests/new)。 请确保包含与帐户关联的电子邮件、当前名称和将来使用的用户名。

若要更改显示 **名称**，请登录到 altvr.com 右上角的配置文件图标"用户配置文件"，"编辑"，然后更新"显示名称"。

**我的菜单为空，无法注册，该怎么办？**

如果未显示"你"菜单，重启 AltspaceVR 客户端应可解决此问题，以便可以登录。 确保 VR 头戴显示设备和 2D 监视器已插入同一 GPU。 如果问题 [仍然存在，](https://help.altvr.com/hc/requests/new) 请与技术支持联系。

## <a name="microsoft-account-integration-faq"></a>Microsoft 帐户集成常见问题解答

**这是否更改了 AltspaceVR 用户名，或者我的详细信息在名称标记中的显示方式？**

将 AltspaceVR 帐户与 Microsoft 帐户链接不会更改公开显示的用户名或名字。 它只会更改帐户登录凭据 (你专用。) 你仍然可以完全控制姓名、用户名和个人资料图片在 AltspaceVR 中的显示方式。

**是否仍可切换帐户？此选项不再位于"设置"下。**

是的，我们重新修饰了一下。 现在，可以在"主菜单"&"我的帐户"下> **帐户>帐户**

**我已有 Microsoft 帐户吗？**

你可能已有 Microsoft 帐户。 你是否在 Xbox 上玩"Mine"、游戏、订阅 Office 365，或者定期使用任何其他 Microsoft 产品或服务？ 如果是这样，你可能已有 Microsoft 登录名！ 可以在此处检查电子邮件是否已绑定到现有[MSA。](https://login.live.com/login.srf?wa=wsignin1.0&rpsnv=13&ct=1610764342&rver=7.0.6738.0&wp=MBI_SSL&wreply=https:%2F%2Faccount.microsoft.com%2Fauth%2Fcomplete-signin%3Fru%3Dhttps%253A%252F%252Faccount.microsoft.com%252F%253Frefp%253Dsignedout-index&lc=1033&id=292666&lw=1&fl=easi2)

**这是否意味着我需要使用 Outlook 创建新的电子邮件地址？**

不知道。 可以使用当前电子邮件地址设置 Microsoft 帐户，即使该帐户未托管在 Microsoft 中。  (例如：G-Mail、Yahoo 等) 。 如果使用与经典 AltspaceVR 帐户共享同一电子邮件的 Microsoft 凭据登录，则登录后系统会提示你链接帐户。 按照提示登录到经典帐户，Altspace 和 Microsoft 帐户将进行链接。

**这是否花费资金？**

不！ 创建 Microsoft 帐户是免费的。 使用 AltspaceVR 也是免费的。 只需记住，你的 Microsoft 帐户已链接到其他产品和服务，因此可以使用该帐户购买 AltspaceVR 之外的其他内容，例如订阅 (例如：Office 365) 或特殊商品 (例如：Mine marketplace) 。

**如果在链接帐户时遇到问题，如何？**

如果遇到问题，请始终通过技术支持 [联系我们](https://help.altvr.com/hc/requests/new)。

**如果我有多个 AltspaceVR 帐户，该做什么？**

只能将一个 AltspsaceVR 帐户链接到 Microsoft 帐户。 需要为要连接的每个 AltspaceVR 帐户创建一个单独的 Microsoft 帐户。

**如果忘记了登录详细信息，该做什么？**

如果在连接到 Microsoft 帐户后无法登录到 AltspaceVR，请通过技术支持 [联系我们](https://help.altvr.com/hc/requests/new)。 如果登录 Microsoft 帐户时遇到问题[](https://support.microsoft.com/account-billing/when-you-can-t-sign-in-to-your-microsoft-account-475c9b5c-8c25-49f1-9c2d-c64b7072e735)，请访问此处或[此处](https://support.microsoft.com/account-billing/how-to-help-keep-your-microsoft-account-safe-and-secure-628538c2-7006-33bb-5ef4-c917657362b9)，获取故障排除提示和资源的列表。

**是否必须连接到我的 Microsoft 帐户？**

别担心，你有一些时间进行更新。 我们将逐步推出 Microsoft 帐户集成。 但是，随着时间的推移，我们正在处理中的一些主要新功能将仅适用于已将 AltspaceVR 帐户链接到其 Microsoft 帐户的个人。 这是因为这些功能与票证 (类似，例如，) 需要非常可靠的安全措施。 有关详细信息，请参阅 [我们的博客](https://altvr.com/microsoft-account-integration)。

**我的隐私和数据呢？**

服务条款或[隐私策略](../community/terms-of-service.md)[没有变化](https://privacy.microsoft.com/privacystatement)。 Microsoft 和 AltspaceVR 仍致力于保护你的隐私。 它们还保持承诺，以确保在数据和信息方面保持完全控制。 可在此处了解有关隐私的 Microsoft 核心[原则。](https://privacy.microsoft.com) 还可以在此处请求有关 Microsoft 关于你的 Microsoft 帐户的个人数据 [的详细信息](https://www.microsoft.com/concern/privacyrequest-msa)。 最后，如果你有有效的 AltspaceVR 帐户，可以请求数据的副本，并请求随时删除数据。
